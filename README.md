# trillion-client
Trillion API client
[Doc](https://trillion-eindhoven.xlab.si/api/docs/)

## Install
```
$ git clone https://github.com/joosthoeks/trillion-clients.git
$ cd trillion-clients
$ [sudo] pip3 install -e .
```

## Update
```
$ cd trillion-clients
$ git pull
```

## Examples / Run / Test
```
$ cd examples
$ python3 run.py
```
